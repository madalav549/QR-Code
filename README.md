# Frontend Mentor - QR code component

This project was my very first challenge on Frontend Mentor. It focuses on creating a clean, responsive card component that centers perfectly on the screen using modern CSS layout techniques.

## Built with

- Semantic HTML5
- CSS Flexbox
- Media Queries
- Google Font(Outfit)

## Key learnings

- **Layout Centering:** I learned how to use `display: flex` with `justify-content: center` and `align-items: center` on the `body` element.
- **Responsive Adjustments:** I discovered how to use`@media` queries to change the width from 30% (Desktop) to 90% (Mobile) to keep it readable.
- **Box Model:** I practiced how `padding` affects the spacing inside the `main-container` to keep the content balanced.
